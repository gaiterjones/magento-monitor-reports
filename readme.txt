MonitorReports

v0.03 TEST Module

This Magento module monitors the var/reports folder for new files and sends email alerts when new reports
are detected. Runs as part of the Magento Scheduled tasks and requires Magento cron.shp script to run periodically.


To install the module, copy the app folder to your magento store app folder.
Refresh your magento cache.
Logout from admin, and login again.


To configure

Enter a valid email address.
